# Memory and Pulse Integration

Mobius Voice sends structured voice intelligence into Mobius Memory and metric signals into Mobius Pulse.

## Memory Outputs

- Raw transcript reference
- Clean transcript
- Speaker map
- Meeting summary
- Decisions
- Action items
- Voice confidence
- Language and dialect context

## Pulse Outputs

- VCI
- speaker verification rate
- transcription confidence
- meeting intelligence completion rate
- action extraction count

## Rule

Raw audio and raw transcripts may be preserved, but speaker identity must be consent-based before becoming trusted organizational memory.

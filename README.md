# Mobius Voice

Voice capture, transcription, speaker intelligence, and multilingual intake layer for the Mobius Organizational Intelligence Platform.

## Department

Mobius Voice belongs to Sonic Department, the Audio Intelligence Division.

## Mission

Transform every spoken interaction into verified organizational intelligence.

## Core Capabilities

- Audio capture
- Raw transcription
- Clean transcription
- Multilingual intake
- Dialect-aware understanding
- Speaker diarization
- Consent-based speaker verification
- Meeting intelligence
- Voice-to-task
- Voice Confidence Index
- Structured output to Mobius Router
- Memory-ready output for Atlas Core

## Boundary

Sonic handles audio. Mobius Router handles routing. Atlas Core handles verification, memory, reasoning, and knowledge graph integration.
